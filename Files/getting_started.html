Part 1: Accounts

Create accounts, bookmark, and explore each of the following platforms

  • Github - This is a platform that stores your code and manages version, allowing you to collaborate or roll back code if needed.
  • CodePen - This is a platform that allows you to quickly experiment with code and share experiments (or 'Pens'). Go here whenever you want to try something out quickly
  • FreeCodeCamp - This is a comprehensive curriculum that will help you learn web development from the beginning
  • Sign up for Slack and say hello!

Part 2: Installations

Install the following tools on your Mac and make sure you are on the guest wifi.

  • Atom - This is the text editor that you will use for writing code and saving files
  • iTerm - This is an application that makes working in the terminal easier - it looks a little better than the default Terminal program
  • Install Chrome and make it your default browser
  • Install and setup GitHub
    1. Open the terminal
    2. Install Homebrew by copying and pasting in the terminal:

      /usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)

    3. Install Git by copying and pasting in order in the terminal:
      • brew update
      • brew install git
    4. Check for existing SSH keys(one way to connect Git and Github) by following the instructions here
    5. If no SSH keys are found, follow the instructions here
    6. Add the SSH key to your account by following the instructions here
    7. And you're done!

Part 3: Practice

  • Use the 'Spotlight Search' to open programs quickly (cmd + shft)
  • Practice using the terminal as much as possible. Once you've installed iTerm, use that instead of the default Terminal program: Type (cmd + shft), then start typing 'iTerm' to find and open the program.
  • Start doing exercises on FreeCodeCamp and keep going!
  • Create experiments on Codepen with some HTML and CSS experiments (try things that you learn from Free Code Camp)
